# 初中数学几何画板 - 后续开发计划

针对您提到的初中数学专用工具集，我们将按照以下阶段进行实现：

## 1. 优化工具栏图标 (立即执行)
解决当前“两个圆圈”的困惑：
- **Point (点)**: 图标改为 `CircleDot` (带点的圆) 或 `Target`，明确区分于圆。
- **Circle (圆)**: 保持 `Circle` (空心圆) 图标。
- **Perpendicular (垂线)**: 保持或优化图标。

## 2. 测量工具集 (Measurement Tools)
实现实时测量与显示，这是几何教学的核心：
- **距离/长度**: 点击两个点或一条线段，显示长度标签。
- **角度**: 点击三个点（顶点在中间）或两条线，显示角度值和弧线标记。
- **面积**: 点击多边形或圆，显示面积。
- **动态更新**: 当图形被拖拽改变时，测量数值实时更新。

## 3. 函数图像模块 (Function Graphs)
实现初中核心函数绘制：
- **输入面板**: 提供解析式输入框 (如 `y = kx + b`, `y = ax^2 + bx + c`)。
- **参数控制**: 识别解析式中的参数 (k, b, a...) 并自动生成滑动条，支持动态变换演示。
- **坐标系增强**: 完善坐标轴数值标记和网格对齐。

## 4. 平面几何教学工具
- **勾股定理演示**: 提供预制的“勾股方图”生成工具，或通过测量面积验证 $a^2+b^2=c^2$。
- **全等/相似验证**: 选择两个三角形，系统自动计算边长比和角度，判断是否全等或相似，并给出判定依据 (SSS, SAS, etc.)。

## 5. 课件制作与演示
- **步骤录制**: 记录作图顺序，支持“下一步”播放。
- **文本与公式**: 集成 LaTeX 公式渲染，方便添加数学说明。
